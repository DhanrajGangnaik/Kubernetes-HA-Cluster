# Prerequisites

## Infrastructure

- Proxmox environment
- VMs for Talos nodes
- VM for Rancher management cluster
- DNS server
- edge NGINX proxy
- workstation with kubectl, talosctl, helm

## Tooling

- kubectl
- talosctl
- helm
- ssh
- curl
- jq
- yq

## Networking

- static IP assignments
- DNS entries
- edge proxy routing
- connectivity between management and workload layers
